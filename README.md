# JACOBO
------
Simple webcrawler desarrollado en python.

Jacobo obtiene, enlaces en un sitio web que incluyen:
- Enlaces externos
- Archivos css
- Archivos js
- Imagenes
- Links de formularios
- Correos electronicos

### Uso
```
$ python jacobo.py http://misitioweb.com
```
